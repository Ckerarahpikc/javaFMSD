# JavaFMSD (File Management & Storage Design)

![Version](https://img.shields.io/github/v/release/Ckerarahpikc/javaFMSD)
![Java](https://img.shields.io/badge/Java-17+-blue)
![JavaFX](https://img.shields.io/badge/JavaFX-Supported-orange)
![Maven](https://img.shields.io/badge/Built_with-Maven-C71A36?logo=apachemaven&logoColor=white)

A JavaFX desktop application designed to manage objects and handle structured file storage.

## 📦 Download the Latest Release
You do not need to compile the code or use any sort of IDE to run this application.
1. Go to the [Releases page](../../releases/latest).
2. Download the latest `.jar` or `.zip` file from the **Assets** section.
3. Make sure you have [Java 17 or higher](https://adoptium.net/) installed on your system.
4. Double-click the `.jar` file to launch the application, or run the following command in your terminal:
   ```bash
   java -jar javaFMSD.jar
   ```
## 🪶 Features 🪶
* **Management:** Tracks and manages life of stored objects.
* **File Storage:** Different read/write operations with a local storage system.
* **GUI:** Straighforward approach to style unique but simple rows and columns.

## 🛠️ Requirements 🛠️
To run the pre-compiled release, you need:
* IDE, PC, Internet (optional).
* Java Runtime Environment (JRE) 17 or higher.

## 🧧 Building from Source (For Developers) 🧧
If you want to view the code, run it in your own environment (like LazyVim or IntelliJ), or contribute, follow these steps:

1. **Clone the repository:**
```bash
	git clone git@github.com:Ckerarahpikc/javaFMSD.git
```

2. **Navigate to the directory:**
```bash
	cd javaFMSD
```

3. **Build the project using Maven:**
```bash
	mvn clean install
```

4. **Run the application:**
```bash
	mvn javafx:run
```
